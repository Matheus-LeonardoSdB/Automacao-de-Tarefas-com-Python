# Automação de Tarefas com Python

Projeto desenvolvido para automatizar tarefas repetitivas utilizando Python e a biblioteca PyAutoGUI.

## Objetivo

Reduzir o tempo gasto em atividades manuais por meio da automação de processos executados no computador.

## Tecnologias Utilizadas

* Python
* PyAutoGUI

## Funcionalidades

* Automatização de ações de teclado e mouse.
* Execução sequencial de tarefas pré-definidas.
* Redução da necessidade de interação manual durante processos repetitivos.

## Como Executar

1. Instale o Python.
2. Instale as dependências necessárias:

```bash
pip install pyautogui
```

3. Execute o arquivo principal:

```bash
python main.py
```
